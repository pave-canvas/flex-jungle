Contact Forms

Contact Form is a plugin (addon) for Page Builder with 20+ options.
It comes with 2 predefined styles shown in examples below: Default and Dark.

“Default” Style - With Captcha

Contact Form with 50% width for input boxes (except “Message”)

“Default” Style - Without Captcha

Without “Captcha”, with “Custom Button”, using custom color and text.

Dark Style

“Dark Style” is for darker background environment.

Dark Style

“Dark Style” is for darker background environment.
Various column sizes can make combination of the fields (columns).

Default Style - With Captcha

“Default Style” has full width for input boxes.

“Dark” Style - All Fields

“Dark Style” with all fields enabled and “custom” styled submit button with full width.

Default Style - With Captcha

With different styles for “Submit” buttons.

Contact Form + Google Map Examples

Combination with another Page Builder Addon - Google Map you can create great Contact Page .
